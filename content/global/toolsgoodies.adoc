== Section 1: Developer Tooling & Testing
Tools and frameworks that support the development and testing process

*Adopt*

- http://www.jetbrains.com/idea[IntelliJ IDEA] IntelliJ IDEA, the one and only IDE you will ever need!
- https://playwright.dev[Playwright] Industry standard for end-to-end testing, offering superior reliability and features compared to alternatives
- https://www.testcontainers.org/[Testcontainers] Essential for modern integration testing, especially in microservices environments
- https://storybook.js.org/[Storybook] De facto standard for component development and documentation in modern frontend development
- https://www.cypress.io/[Cypress] Mature and widely adopted for frontend testing, though consider Playwright for new projects

*Assess*

- https://insomnia.rest/[Insomnia] Solid API development tool, though consider if your team needs more enterprise features
- http://pitest.org/[Pitest : Real world mutation testing] PIT is a state of the art mutation testing system, providing gold standard test coverage for Java and the jvm. Its fast, scalable and integrates with modern test and build tooling.
- http://infinitest.github.io/[infinitest] Useful for continuous testing, but evaluate if it fits your development workflow

*Hold*

- http://seleniumhq.org/[Selenium] While still used, newer tools like Playwright offer better developer experience
- https://github.com/powermock/powermock/[PowerMock] Use with caution, prefer proper dependency injection and design patterns

*Drop*

- http://sqlfiddle.com/[SQL Fiddle] Better alternatives exist for database experimentation
- http://www.balsamiq.com/[Balsamiq mockups] More modern UI/UX design tools available now

== Section 5: Development Languages & Frameworks
Programming languages, frameworks, and related tools

*Adopt*

- http://asciidoctor.org/[Asciidoctor] Toolchain for converting AsciiDoc content to HTML5, DocBook 5 (or 4.5) and other formats.

*Assess*

- http://angularjs.org/[Angular] Mature framework with strong enterprise adoption
- http://www.gradle.org/[Gradle Build Tool] Standard build tool for Java ecosystem
- https://postcss.org[PostCSS] A tool for transforming CSS with JavaScript.
- https://lernajs.io[Lerna] Evaluate against newer monorepo tools
- http://www.eclipse.org/aspectj/[AspectJ] Still relevant for specific use cases

- http://www.querydsl.com/[QueryDSL] Evaluate against modern alternatives
- http://juel.sourceforge.net/[JUEL] Limited use cases in modern development

*Drop*

- http://ocpsoft.org/prettyfaces/[PrettyFaces] Outdated URL handling for JSF
- http://arquillian.org/[arquillian.org] Arquillian is framework to write tests for JEE applications.
- http://mogwai.sourceforge.net/[Project Mogwai] Better alternatives are available, but I still love this project
